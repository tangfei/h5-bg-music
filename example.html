背景音乐播放示例

说明：此页面演示了如何在普通HTML页面中使用背景音乐播放功能。

在移动设备上打开此页面，背景音乐应该会自动播放（特别是在微信浏览器中）。

// 使用示例

// 初始化背景音乐
const bgMusic = BgMusic.init({
  src: 'http://music.163.com/song/media/outer/url?id=1814767154.mp3',
  loop: 999,
  controls: true,
  customStyle: {
    right: '20px'
  }
});

// 播放音乐
bgMusic.play();

// 暂停音乐
bgMusic.pause();

// 获取播放状态
const state = bgMusic.getState(); // 'play' 或 'pause'

// 销毁音乐实例
bgMusic.destroy();

注意事项

  • 音频文件必须可以通过HTTPS访问且没有跨域问题
  • 在某些浏览器中，首次播放可能需要用户交互（点击等）
  • 建议使用短链接音频文件，以提高加载速度
  • 在生产环境中，请使用自己的音频文件CDN
